# Portfolio Website

A modern, animated portfolio website built with Next.js 14, Framer Motion, and Tailwind CSS.

## Features

- Smooth animations with Framer Motion
- Responsive design
- Dark theme with gradient accents
- Sections: Hero, About, Skills, Projects, Contact
- Contact form
- Resume download

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

Update the following:
- Replace "Your Name" in `components/Hero.tsx`
- Update social links in `components/Hero.tsx`
- Add your projects in `components/Projects.tsx`
- Update skills in `components/Skills.tsx`
- Update contact info in `components/Contact.tsx`
- Add your resume PDF to `public/resume.pdf`

## Build for Production

```bash
npm run build
npm start
```

## Deploy

Deploy easily on Vercel, Netlify, or any platform that supports Next.js.
